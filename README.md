Introduction

Attendance management is a critical task in educational institutions. Traditional paper-based systems are slow, difficult to manage, and prone to errors or data loss.
This project is an Android-based Attendance Management System designed to help teachers digitally record, store, and view student attendance using Firebase. The system provides secure authentication, real-time data synchronization, and easy access to attendance reports.
